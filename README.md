# SCM_SGP_IV - Supply Chain Management Tool  A comprehensive Supply Chain Management (SCM) tool for chemical manufacturing. 
## Features 
### 1. Raw Material Management 
- Purity tracking
- Approved suppliers list
- Hazard class classification
- Storage conditions monitoring
  
### 2. Supplier Management
- Approved vendors list
- Certifications tracking
- Past quality issues history

### 3. Batch Management
- Raw material acquisition tracking
- Production date tracking
- Buyer information
